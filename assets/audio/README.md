# Audio Assets

Place your audio files here. Phaser tries .ogg first, then .mp3.

Required files:
  intro.ogg / intro.mp3         — music for the intro screen
  gameplay.ogg / gameplay.mp3   — music during the race
  start.ogg / start.mp3         — SFX: car slides in on intro
  checkpoint.ogg / checkpoint.mp3 — SFX: crossing the finish line
  finish.ogg / finish.mp3       — SFX: goal reached
  timeup.ogg / timeup.mp3       — SFX: time ran out

If a file is missing Phaser silently skips it — no crash.
